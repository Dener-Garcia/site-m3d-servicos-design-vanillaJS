Domine o terminal do Linux

Nesse post quero te ensinar a dominar totalmente os comandos do linux, vamos aprender os comandos mais simples até os mais avançados como configuração de rede, acesso de usuários e muito mais.

Já pega a dica... No final do post tem uma lista com todos os comandos utilizados nesse post, então se você precisar de algum comando específico basta dar uma olhada na lista.

Porque dominar o terminal?

Em linhas básicas, o terminal do Linux, também conhecido como shell, é um programa que recebe os comandos do usuário a partir do teclado e repassa-os às camadas de baixo nível do sistema operacional. Logo, podemos defini-lo como um mecanismo de software que intermedeia a interação do usuário com o Linux.

Manual de ajuda - "man"

O primeiro comando que vamos aprender é o man ele serve para mostrar o manual de algum outro comando que você está utilizando ou que queria saber como funciona.

man ls

Ajuda rápida - "help"

Outro comando de ajuda é o "xxx" --help semelhante ao "man" mas com menos informações, sendo bastante útil para rápidas pesquisar de algum parâmetro que desejamos saber como funciona.

ls --help

Usuário Root - "sudo -i"

Algumas vezes precisamos nos tornar o usuário root do sistema o que tem permissão para alterar arquivos do próprio Linux para isso usamos o sudo -i Lembre-se de tomar cuidado com suas ações estando nesse modo.

sudo -i

Torna-se outro usuário do sistema - "su"

Para se tornar outro usuário do sistema utilize o su "xxx" será necessário ter a senha desse usuário para ter acesso as suas pastas.

su OutroUsuário

Criar um novo usuário - "sudo adduser"

Somente o usuário sudo pode criar novos usuários através do comando sudo adduser "xxx", não há limites para de quantidade de usuários que podem ser criados.

sudo adduser NovoUsuário

Alterar senha de um usuário - "passwd"

Para trocar a senha do seu usuário atual pasta utilizar o comando passwd seguido da nova senha que deve cumprir alguams regras. para mudar a senha de outro usuário basta adicionar o nome dele a frente do comando passwd "xxx".

passwd sudo passwd OutroUsuário

Excluir um usuário - "sudo userdel"

Somente o usuário sudo pode deletar os usuários com o seguinte comando sudo userdel "xxx", Há alguns parâmetros interessantes para se usar com esse comando, como o -r para remover o diretório de home desse usuário.

sudo userdel NovoUsuário

Modificando o grupo de um usuário - "sudo usermod -aG"

Com o comando sudo usermod "xxx" "yyy" junto com os parâmetros -aG permitem adicionar um usuário a outros grupos, sem remove-lo do grupo atual, a frente do parâmetro insira o nome e logo depois o grupo novo do usuário.

sudo usermod -aG NomeUsuario NovoGrupoUsuario

Bloquear e desbloquear um usuário - "sudo usermod -L ou -U"

Outra possíbilidade do comando sudo usermod -L "xxx" é a realização do bloqueio de um usuário através do parâmetro -L, e para liberar o usuário use o comando sudo usermod -U "xxx" adicionando parâmetro -U a frente do usermod.

sudo usermod -L NomeUsuario sudo usermod -U NomeUsuario

Criando grupo de usuários - "sudo groupadd"

Um grupo pode ser utilizado para categorizar múltiplos usuários com permissões diferentes, para tal basta usar o comando sudo groupadd "xxx" um comando simples mas muito funcional.

sudo groupadd NomeUsuário

Mostrar o diretório atual - "pwd"

Algumas vezes podemos ficar confusos de onde realmente estamos dentro do terminal, para isso temos o comando pwd print work directory, ele mostra exatamente em qual diretório você esta.

pwd

Listar arquivos e diretórios - "ls"

O comando ls exibe as pastas e arquivos de um diretório, podendo ser aquele que você está no momento, ou um outro diretório que você deseja saber, basta adicionar o caminho completo a frente, com o comando ls /xxx/xxx dependendo da pasta que se deseja listar os diretórios pode ser necessário o uso do sudo antes do comando.

ls ls /home/NomeUsuario/Videos

Principais parâmetros usados com ls - "ls -a e -h"

Há muitos parametros úteis para o ls, por exemplo para ver pastas invisíveis use o parâmetro "-a", para ver o tamanho das pastas ou arquivos e o tipo de permissionamento que cada um possui usamos o parâmetro -lh.

ls -a /home/NomeUsuario ls -lh /home/NomeUsuario

Alterar o dono de um arquivo - "sudo chown"

Com o comando sudo chown "xxx" "yyy" é possível alterar o dono de um arquivo ou até mesmo de um diretório inteiro.

sudo chown root NomeArquivo

Alterar o grupo do dono de um arquivo - "sudo chown :"

O comando sudo chown "xxx":"yyy" "zzz" é possível alterar o grupo do arquivo e definir ou não um novo dono.

sudo chown root:adm NomeArquivo

Alterar as permissões de arquivos - "sudo chmod -R"

O comando sudo chmod "xxx" "yyy" é usado para lidar com permissões de arquivos do sistema Linux, com esse comando o dono do arquivo (ou seja, o usuário que o criou), pode mudar as permissões dos demais usuários para aquele arquivo, use o -R para redefinir os arquivos dentro de um diretório também.

sudo chmod -R 751 NomeArquivo

Criar diretórios - "mkdir"

Para criar um diretório use o comando mkdir "xxx" dentro de algum diretório que você esteja ou digitando o endereço completo do diretório, também é possível criar sub-diretórios dentro do novo diretório, basta usar o parâmetro"-p" a frente do "mkdir".

mkdir NovoDiretorio mkdir /home/NomeUsuario/Documents/NovoDiretorio mkdir -pv /home/NomeUsuario/Documents/NovoDiretorio/NovoSubDiretorio/NovoSubSubDiretorio

Mudando de diretório - "cd"

Um dos comandos mais utilizados no terminal é o cd "xxx" ele serve para trocar você de diretório assim como você faria numa interface abrindo uma pasta depois a outra e voltaria na primeira pasta, você faz essas ações com esse comando.

O comando "cd" tem vários parâmetros úteis, ele sozinho te joga de volta na sua "home", usando cd .. você volta em um diretório anterior, colocando o endereco completo você pode entrar em qualquer diretório que deseja de maneira rápida.

cd NomeDiretorio cd .. cd /NomeDiretorio1/SubDiretorio1 cd ../NomeDiretorio2/SubDiretorio2

Copiando arquivos e diretórios - "cp"

O comando cp "xxx" "yyy" serve para copiar um arquivo ou diretório com pastas e subpastas de um local para o outro, no momento da cópia também é possível alterar o nome do arquivo colocando um novo nome após o caminho de destino, outro detalhe é que quando existe arquivos dentro das pastas é necessário usar alguns parâmetros para forçar essa cópia.

cp NomeArquivo /caminho/destino cp NomeArquivo /caminho/destino/NovoNomeArquivo.extensao cp -rvp NomeDiretório /caminho/destino cp -rvp NomeDiretório /caminho/destino/NovoNomeDiretório
